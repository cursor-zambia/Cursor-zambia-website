# Cursor Zambia Community — Landing Page

Static single-page site (HTML/CSS/JS) with:
- Cursor-like dark styling + neon accent
- Sticky “tabs” nav that highlights as you scroll
- Scroll-telling section (sticky panel + steps)

## Preview locally (Windows)

If you have Python installed:

```bash
python -m http.server 5173
```

Then open `http://localhost:5173` in your browser.

## Customize

- Edit content in `index.html`
- Theme and layout in `styles.css`
- Interactions (tabs, reveal, scroll-telling, contact-copy) in `main.js`
